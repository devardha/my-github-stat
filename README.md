# My GitHub Stat

## Endpoint
- `/api/stats/devardha/languages` - Top languages stats

### Parameters
- `type` - default, compact
- `theme` - default, panda

## Example
- `/api/stats/devardha/languages?theme=panda`
- `/api/stats/devardha/languages?type=compact`
- `/api/stats/devardha/languages?type=compact&theme=panda`
